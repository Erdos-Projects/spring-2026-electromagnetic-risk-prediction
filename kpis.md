# Primary KPIs
1. Event recall (did we predict a storm would occur when there was a storm). Note, real world variability will limit how high this number can truly be.

# Secondary KPIs
1. Minimum event precision (we do not continuously say there is a storm when there is not). In other words, the F1-Score for both classes needs to be balanced.
2. Lead time --- longer term predictions are more helpful, but also tend to be less accurate
